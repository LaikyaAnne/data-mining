What is an array?
An array is a ordered collection of homogenious(similar type) elements stored in continueous memory locations.

All the elements of an array refered through a single name with index value.

Declaration:
		Syntax:
				datatype arr_name[size];
		Ex:
				int marks[6];				=> 6 x 4 => 24 bytes
				float heights[10];			=> 10 x 4 => 40 bytes
				char ch[5];					=> 5 x 1 => 5 bytes
				double salaries[100];		=> 100 x 8 => 800 bytes
				
Initialization:
==============
It is a process of assigning values into an array is called initialization of an array.
Types of initialization:
1) Compiletime initialization:
   Assigning the values into an array at the time of declaring the array is called compile time initialization.
   Syntax:
			datatype arr_name[size]={value1,value2,......};
			or
			datatype arr_name[]={value1,value2,....};
			
   Ex:
			int marks[6]={45,65,78,76,67,89};
			
program:
#include<stdio.h>
int main()
{	int marks[6]={67,85,98,87,76,87};
	// Print the array elements without using any loop
	printf("%d\n",marks[0]);  
	printf("%d\n",marks[1]); 
	printf("%d\n",marks[2]); 
	printf("%d\n",marks[3]); 
	printf("%d\n",marks[4]); 
	printf("%d\n",marks[5]);  
	//Print the array using for loop
	int i;
	for(i=0;i<6;i++)
	{
		printf("%d ",marks[i]);
	}
	return 0;
}

2) Runtime initialization:
   Assigning the values into an array at the time of  program execution is called runtime initialization.
   Example:
		int arr[size];
		for(i=0;i<size;i++)
		{
		     scanf("%d",&arr[i]);
		}

Program:
#include<stdio.h>
int main()
{
	int n;
	printf("Enter the size of the array\n");
	scanf("%d",&n);    // n is the size of the array

Write program to perform clock-wise circular rotation of elements in an array by one step.
Input:
5
11 22 33 44 55

Output:
55 11 22 33 44

Code:

#include<stdio.h>
int main()
{
	int n,i;
	scanf("%d",&n);
	int arr[n];
	for(i=0;i<n;i++)
	   scanf("%d",&arr[i]);
	printf("Array Elements before Circular Rotation\n");
	for(i=0;i<n;i++)
		printf("%d ",arr[i]);
	printf("\n");
	int temp=arr[n-1];
	for(i=n-1;i>=1;i--)
	{
		 arr[i]=arr[i-1];
	}
	arr[0]=temp;
	printf("Array Elements after Circular Rotation\n");
	for(i=0;i<n;i++)
	   printf("%d ",arr[i]);
	return 0;
}
